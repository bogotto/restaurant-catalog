# Онлайн-каталог продукции ресторана

Веб-приложение — онлайн-каталог блюд ресторана. Посетитель просматривает меню
по категориям, ищет блюда, изучает состав и КБЖУ, добавляет блюда в корзину и
оформляет заказ. Администратор управляет категориями и блюдами через панель
управления.

## Стек

- **Backend:** Python, Django
- **Frontend:** HTML, CSS (шаблоны Django)
- **База данных:** SQLite (локально)
- **Pillow** — работа с изображениями блюд

## Возможности

- Каталог блюд с разбивкой по категориям
- Карточка блюда: описание, цена, вес, фото
- **КБЖУ** (калорийность, белки, жиры, углеводы) у каждого блюда
- **Поиск по сайту** — по названию и описанию блюд
- **Регистрация и вход** пользователя (гостя)
- **Корзина** на основе сессии (работает и для гостей)
- **Оформление заказа** и раздел **«Мои заказы»** для зарегистрированных пользователей
- **Лайки** блюд (гость может оценивать понравившиеся блюда)
- **Акции ресторана** — отдельная страница со скидками по дням недели
- Страница **«О ресторане»** с галереей фотографий
- Поддержка: контакты и форма обратной связи (обращения сохраняются в админке)
- Панель администратора для управления каталогом, заказами, акциями, фото и обращениями
- Адаптивный интерфейс (десктоп и телефон)

## Структура проекта (модульная архитектура)

```
restaurant/   — настройки и маршрутизация проекта
catalog/      — каталог: категории, блюда, КБЖУ, поиск
accounts/     — регистрация, вход, выход
cart/         — корзина, заказы, «Мои заказы»
pages/        — акции и страница «О ресторане» с галереей
static/css/   — оформление
media/dishes/ — загружаемые фотографии блюд
```

## Как запустить локально

```bash
git clone <repo-url>
cd restaurant-catalog

python -m venv venv
# Windows:
venv\Scripts\activate
# Linux/macOS:
source venv/bin/activate

pip install -r requirements.txt

python manage.py migrate
python manage.py loaddata sample_dishes        # 8 тестовых блюд с КБЖУ
python manage.py loaddata sample_promotions    # тестовые акции
python manage.py createsuperuser          # создать администратора
python manage.py runserver
```

После запуска открыть:
- Каталог: http://127.0.0.1:8000/
- Админка: http://127.0.0.1:8000/admin/

Фотографии блюд добавляются через админку (поле «Фото» у блюда).
